Math Nerd ▼ 
    Quick Condition Check: 
    Monster Target (T) < Total Cards (C) < 2×Target
    
    Derivation:
    Let X = XYZ Rank, F = Fusion Level
    
    From SEC Requirements:
    ① 2X + F = C (Total Cards)
    ② X + F = T (Monster Target)
    
    Solving Simultaneously:
    From ②: F = T - X
    Substitute into ①:
    2X + (T - X) = C → X = C - T
    
    Then:
    F = T - (C - T) = 2T - C
    
    Valid Solutions Require:
    X > 0 → C - T > 0 → C > T
    F > 0 → 2T - C > 0 → C < 2T
    
    ∴ T < C < 2T
