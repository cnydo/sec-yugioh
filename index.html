TLDR

\textup(Monster Target) \( T < C < 2T \)

Derivation

Let \( X \) = XYZ Rank, \( F \) = Fusion Level

From SEC Requirements:

\[ \begin{cases} 2X + F = C & \text{(Total Cards)} \\ X + F = T & \text{(Monster Target)} \end{cases} \]

Solving Simultaneously:

From ②: \( F = T - X \)

Substitute into ①:

\[ 2X + (T - X) = C \implies X = C - T \]

Then:

\[ F = T - (C - T) = 2T - C \]

Valid Solutions Require:

\[ \begin{aligned} X > 0 &\implies C - T > 0 \implies C > T \\ F > 0 &\implies 2T - C > 0 \implies C < 2T \end{aligned} \] \[ \therefore T < C < 2T \]
